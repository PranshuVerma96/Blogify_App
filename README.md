# 🚀 Blogify - Blogging Web App

Blogify is a full-stack blogging application built using Node.js, Express, MongoDB, and EJS. It allows users to create, read, and manage blog posts with authentication and image uploads.

---

## ✨ Features

* 🔐 User Authentication (Signup/Login)
* 📝 Create & View Blogs
* 🖼️ Upload Cover Images (Multer)
* 💬 Comment on Blogs
* 🎨 Dynamic UI using EJS

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB (Mongoose)
* EJS
* Multer
* JWT (JSON Web Token)

---

## 📂 Project Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/blogify.git
cd blogify
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the project

```bash
npm start
```

---

## ⚠️ Environment Variables

Create a `.env` file in the root folder and add:

```env
MONGO_URL=mongodb://localhost:27017/blogify
```
